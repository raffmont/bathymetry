Bathymetry demo

This project turns EMODnet bathymetry DTM tiles into vector contours and raster overlays (bathymetry, hillshade, slope). The demo server serves those outputs as MBTiles and exposes a simple UI to preview them.

Quick start

  1. Generate tiles using config.json or examples/config-demo.json.
  2. Run python demo/server.py (uses demo/config.json).
  3. Pick a visualization below.

Choose a viewer

  • MapLibre viewer — vector + raster overlay selection.
  • OpenLayers viewer — vector + raster overlay selection.

What the demo shows

  • Vector contour lines (MVT)
  • Raster bathymetry tiles (PNG8)
  • Derived hillshade and slope overlays (PNG8)
